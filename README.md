# adigrams

Draw lines in ascii...
    
```
    +_________+
    !         !   for real
    !  ADI +--+---|------+
    !      |:)!   |      |
    +______+__+  GRAMS   |
           |             |
           +-- oh yeah --+
```

...and get Unicode box drawing characters:

```
    ┏━━━━━━━━━┓
    ┃         ┃   for real
    ┃  ADI ┌──╂───│──────┐
    ┃      │:)┃   │      │
    ┗━━━━━━┿━━┛  GRAMS   │
           │             │
           └── oh yeah ──┘
```
